 ## Editkaro.in Portfolio Website

## 📌 Description

This project is a responsive portfolio website developed for **Editkaro.in**, a social media marketing and video editing agency. 
The website showcases different categories of video editing work through an interactive portfolio with category-based filtering. 
It features a modern user interface, responsive design, smooth animations, and an engaging browsing experience.

## 🚀 Features

- Responsive landing page
- Modern navigation bar
- Hero section
- About section
- Interactive portfolio gallery
- Category-based filtering using JavaScript
- 12 portfolio cards
- Hover animations
- Contact section
- Responsive design
- Smooth scrolling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Font Awesome
